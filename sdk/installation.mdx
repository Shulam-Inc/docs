# Installation

Shulam provides two packages depending on your use case.

## Server SDK

For Express/Node.js backends that accept payments:

```bash
npm install @shulam/sdk
```

**Subpath exports:**

| Import | Purpose |
|--------|---------|
| `@shulam/sdk/express` | `requirePayment()` middleware |
| `@shulam/sdk/compliance` | `Shulam.compliance.screen()` |
| `@shulam/sdk/webhooks` | Webhook signature verification |
| `@shulam/sdk` | Core types and utilities |

## Full Stack / Client SDK

For frontends, full-stack apps, or agent-to-agent payments:

```bash
npm install @shulam/x402
```

**Subpath exports:**

| Import | Purpose |
|--------|---------|
| `@shulam/x402/client` | `createShulamClient()` — auto-402 fetch |
| `@shulam/x402/server` | `createShulamServer()` — multi-endpoint config |
| `@shulam/x402/react` | `useShulamPayment()` hook |
| `@shulam/x402/types` | Shared TypeScript types |
| `@shulam/x402` | Core protocol utilities |

## Requirements

- Node.js 18+
- React 18+ (for `@shulam/x402/react`)
- An EIP-1193 compatible wallet (MetaMask, Coinbase Wallet, etc.)

## Peer Dependencies

Both packages require `viem` for Ethereum interactions:

```bash
npm install viem
```

## Quick Start

### Server (Express)

```typescript
import { requirePayment } from "@shulam/sdk/express";

app.get(
  "/api/data",
  requirePayment({ amount: "5000000", payTo: "0xYourWallet" }),
  (req, res) => {
    res.json({ data: "paid content", payer: req.payment.payer });
  }
);
```

### Client

```typescript
import { createShulamClient } from "@shulam/x402/client";

const client = createShulamClient({
  facilitatorUrl: "https://api.shulam.io",
  signer: walletSigner,
  walletAddress: "0xBuyerAddress",
  maxPaymentUSDC: 100,
});

const response = await client.fetch("https://merchant.com/api/data");
```
