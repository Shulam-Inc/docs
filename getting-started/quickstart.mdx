# Quickstart

Go from `npm install` to your first USDC payment on Base Sepolia testnet in three steps.

## Prerequisites

- A wallet address to receive payments (any Ethereum-compatible wallet)
- Node.js 18+ installed

## Step 1: Install the SDK

<Tabs>
  <Tab title="Server (Express)">
    ```bash
    npm install @shulam/sdk
    ```
  </Tab>
  <Tab title="Full Stack">
    ```bash
    npm install @shulam/x402
    ```
  </Tab>
</Tabs>

## Step 2: Add a Paywall

### Express

Three lines to protect any route with a USDC paywall:

```typescript
import express from "express";
import { requirePayment } from "@shulam/sdk/express";

const app = express();

app.get(
  "/api/premium",
  requirePayment({ amount: "10000000", payTo: "0xYourWallet" }),
  (req, res) => {
    // req.payment contains { txHash, payer, settled }
    res.json({ data: "premium content", paidBy: req.payment.payer });
  }
);

app.listen(3000);
```

### Next.js

```typescript
import { requirePayment } from "@shulam/sdk/express";

const paywall = requirePayment({
  amount: "5000000", // 5 USDC
  payTo: "0xYourWallet",
});

export async function GET(req: Request) {
  const payment = await paywall(req);
  return Response.json({ data: "premium content", paidBy: payment.payer });
}
```

## Step 3: Test a Payment

Use the client SDK to pay through your paywall:

```typescript
import { createShulamClient } from "@shulam/x402/client";

const client = createShulamClient({
  facilitatorUrl: "https://api.shulam.io",
  signer: yourWalletSigner,
  walletAddress: "0xBuyerAddress",
  maxPaymentUSDC: 100,
});

// Automatically handles 402 responses â€” signs, pays, and retries
const response = await client.fetch("http://localhost:3000/api/premium");
const data = await response.json();
```

### React

```tsx
import { useShulamPayment } from "@shulam/x402/react";

function PayButton() {
  const { pay, paying, lastReceipt, error } = useShulamPayment({
    facilitatorUrl: "https://api.shulam.io",
    signer: walletSigner,
    walletAddress: "0xBuyerAddress",
    maxDailyUSDC: 50,
  });

  return (
    <button onClick={() => pay("http://localhost:3000/api/premium")} disabled={paying}>
      {paying ? "Paying..." : "Access Premium"}
    </button>
  );
}
```
