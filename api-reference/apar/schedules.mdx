## Create Schedule

### Request Body

<ParamField body="invoiceId" type="string" required>Invoice to schedule payment for</ParamField>
<ParamField body="businessId" type="string" required>Business ID</ParamField>
<ParamField body="type" type="string" default="immediate">`immediate`, `scheduled`, or `recurring`</ParamField>
<ParamField body="executeAt" type="string">ISO-8601 execution time (required for scheduled/recurring)</ParamField>
<ParamField body="interval" type="string">Recurring interval: `weekly`, `biweekly`, `monthly`, `quarterly`</ParamField>
